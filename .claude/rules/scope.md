# Rule: Scope Control
## When: Implementing any feature
## Do:
- Follow MVP scope lock (blueprint Section 1)
- Implement only what the active contract specifies
## Don't:
- No PWA, offline sync, push notifications
- No real-time multi-user sync
- No PIN login, barcode, invoice parsing
- No advanced analytics, AI suggestions
- No product/user admin UI
- No Vercel cron
- No new features beyond blueprint
## Why: MVP scope is locked. Ship what's defined, nothing more.
